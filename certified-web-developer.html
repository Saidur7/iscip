Certified Web Developer

Course Description: This course is designed to start you on a path toward future studies in web development and design, no matter how little experience or technical knowledge you currently have. The web is a very big place, and if you are the typical internet user, you probably visit several websites every day, whether for business, entertainment or education. But have you ever wondered how these websites actually work? How are they built? How do browsers, computers, and mobile devices interact with the web? What skills are necessary to build a website? With almost 1 billion websites now on the internet, the answers to these questions could be your first step toward a better understanding of the internet and developing a new set of internet skills.

What you will learn: By the end of this course you’ll be able to describe the structure and functionality of the world wide web, create dynamic web pages using a combination of HTML, CSS, and JavaScript, apply essential programming language concepts when creating HTML forms, select an appropriate web hosting service, and publish your webpages for the world to see.
Course Instructor: Professor Dr. Asif Salam
Program structure: 3 Months - 4 hours online class per week using zoom platform | Total Sessions 48
Course Fees: CAD$ 2000

Who will get benefits: Finally, you’ll be able to develop a working model for creating your own personal or business websites in the future and be fully prepared to take the next step in a more advanced web development or design course or specialization..

Course outline:

  • Computer Studies
  • Course Overview and Website Structure and Hosting
  • Designing Your Own Website: HTML , Advanced HTML, XHTML
  • Introduction to Programming Using JavaScript
  • Websites with Style: CSS Properties, Colors and Fonts
  • Creating HTML Forms
  • Creating Web Applications
  • Web Templates Editing
  • Word Press
  • Data Exchange (HTTP | Websockets)
  • Adobe Photoshop - Graphic Design Modules
  • SEO (Search Engine Optimization)
  • SMO (Social Media Optimization)
  • Advanced Digital Marketing Techniques
